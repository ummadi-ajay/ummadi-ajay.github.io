What Your Child Will Learn

micro:bit & MakeCode Programming

Students learn programming using the BBC micro:bit and Microsoft MakeCode, building logic with visual block-based coding.

  • Basic programming concepts
  • Problem-solving with logic
  • Real-time hardware interaction
Electronics & Sensors

Children explore how electricity works through hands-on experiments involving LEDs, resistors, sensors, motors, and real circuits.

  • Current, voltage & resistance
  • Ohm’s Law (visual & practical)
  • Ultrasonic sensors & servo motors
3D Designing & Printing

Students design real 3D objects that support their electronics projects, helping them connect digital creativity with physical products.

  • 3D shapes & measurements
  • Project enclosures
  • Introduction to 3D printing
Thinking Like an Engineer

Beyond tools, students develop logical thinking, patience, creativity, and confidence through guided trial-and-error learning.

  • Debugging mindset
  • Creative problem solving
  • Presentation & confidence

Session-wise Learning Breakdown (20 Modules)

Our Beginner Program follows a carefully planned session-wise structure, ensuring gradual learning, strong fundamentals, and hands-on confidence.

Module 1: Introduction to Computers & micro:bit

  • What is a computer & microcontroller
  • Introduction to BBC micro:bit
  • LED matrix & buttons exploration
  • First MakeCode program

Module 2: MakeCode Interface & Basic Blocks

  • MakeCode editor overview
  • Basic blocks & event handling
  • Show text & icons on LED display
  • Mini LED animation project

Module 3: Inputs & Sensors on micro:bit

  • Button A & B programming
  • Shake, tilt & motion input
  • Light & temperature sensor usage
  • Reaction-based mini games

Module 4: Variables & Data Storage

  • Understanding variables
  • Creating & updating variables
  • Counters & score tracking
  • Score-based game project

Module 5: Logic & Decision Making

  • If & if–else conditions
  • Comparison operators
  • Real-life decision logic
  • Smart condition-based program

Module 6: Loops & Repetition

  • Forever & repeat loops
  • Nested loops
  • Optimizing code using loops
  • LED animation challenge

Module 7: Functions & Code Reusability

  • Why functions are needed
  • Creating custom functions
  • Calling functions in programs
  • Structured coding project

Module 8: Introduction to Electronics

  • What is electricity
  • Current, voltage & resistance
  • Understanding circuits
  • Basic safety rules

Module 9: Ohm's Law & LEDs

  • Ohm's Law explained simply
  • LED polarity & resistors
  • Brightness control
  • LED lighting project

Module 10: Ultrasonic Sensor

  • Distance measurement concept
  • Ultrasonic sensor wiring
  • Distance-based alerts
  • Smart parking demo

Module 11: Servo Motor Control

  • What is a motor
  • Servo motor angles
  • Micro:bit + servo control
  • Automatic gate project

Module 12: Combining Sensors & Motors

  • Sensor + motor logic
  • Distance-controlled movement
  • Smart automation project

Module 13: Introduction to 3D Designing

  • What is 3D design
  • Basic shapes & scaling
  • Designing name tags

Module 14: 3D Models for Electronics

  • Sensor holders
  • Project enclosures
  • Design optimization

Module 15: Introduction to 3D Printing

  • How 3D printers work
  • Materials & safety
  • Printing student designs

Module 16: Interactive micro:bit Projects

  • Game-based learning
  • Multi-input programs
  • Creative coding challenges

Module 17: Smart Systems

  • Smart alarms
  • Automation logic
  • Real-life problem solving

Module 18: Capstone Project Planning

  • Project idea selection
  • Design & coding plan
  • Component selection

Module 19: Capstone Project Execution

  • Building & testing
  • Debugging & improvements
  • Final project assembly

Module 20: Demo Day & Certification

  • Project presentation
  • Parent demo day
  • Certificate distribution
