Sessions & Curriculum (Advanced)

A rigorous 2-year journey covering AI, Robotics, Embedded Systems, and Product Development.

Module 1: Advanced Python for Robotics

  • Object-Oriented Programming (OOP)
  • Data Structures & Algorithms
  • Multithreading & Asynchronous Coding

Module 2: Computer Vision with OpenCV

  • Image Processing Fundamentals
  • Object Detection & Tracking
  • Face Recognition & Gesture Control

Module 3: Machine Learning & AI

  • Supervised vs Unsupervised Learning
  • Neural Networks & Deep Learning
  • Deploying Models to Edge Devices

Module 4: Introduction to ROS 2

  • ROS Architecture (Nodes, Topics, Services)
  • Publisher/Subscriber Implementation
  • Simulating Robots in Gazebo/Rviz

Module 5: Robot Navigation & SLAM

  • Lidar mapping & Localization
  • Path Planning Algorithms (A*, Dijkstra)
  • Autonomous Navigation Stack

Module 6: Embedded Systems & RTOS

  • Real-Time Operating Systems (FreeRTOS)
  • STM32 / ESP32 Advanced Programming
  • Industrial Communication (CAN, Modbus)

Module 7: IoT & Cloud Integration

  • MQTT & Cloud Protocols
  • AWS IoT / Google Cloud Integration
  • Data Visualization Dashboards

Module 8: Product Development & CAD

  • Advanced 3D Modeling (Fusion 360)
  • Design for Manufacturing (DFM)
  • PCB Design & Fabrication

Module 9: Capstone Research & Proposal

  • Problem Statement & Literature Review
  • Feasibility Study & System Design
  • Technical Proposal Documentation

Module 10: Final Project Build & Defense

  • Hardware Assembly & Firmware Coding
  • Testing, Calibration & Optimization
  • Final Presentation & Review
