Blink an LED with Arduino

The "Blink" project is the most popular beginner project in Arduino. It's often referred to as the "Hello World" of hardware programming because it teaches the very basics of controlling a microcontroller and interacting with electronic components.

Why This Project Matters

It introduces you to how digital pins work, how timing is controlled with code, and how circuits carry current to an output device such as an LED. Without this knowledge, more complex projects like robotics, IoT, or automation will be hard to build.

What You Need
Arduino Uno board
1 LED (any color)
1 resistor (220Ω)
Breadboard & wires
Circuit Setup
  1. Insert the LED into the breadboard.
  2. Connect the longer leg to pin 13 via resistor.
  3. Connect shorter leg to GND pin.
  4. Upload the code via USB.
Code
void setup() {
  pinMode(13, OUTPUT);
}
void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}

Completing this project gives you confidence to move on to sensors, motors, and real-world IoT projects.

Top 5 Sensors for Arduino & Pi

Sensors are the eyes and ears of your IoT projects. They let your microcontroller gather real-world data and respond smartly.

PIR Motion Sensor

Detects movement of humans or animals. Perfect for security alarms and automatic lights.

DHT11 Temp & Humidity

Measures environmental conditions. Common in weather stations and smart homes.

Ultrasonic Sensor

Measures distance using sound waves. Used in obstacle avoidance robots.

IR Sensor

Detects objects or remote signals. Used in line-following robots.

Gas Sensor (MQ)

Detects gases like LPG, CO, or smoke. Essential for safety systems.

By experimenting with these sensors, you'll learn how machines interact with the physical world.
